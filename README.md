介绍

    基于内存key-value的存储服务， 支持多节点备份容灾
    
特点

    强一致性
    高可用性
    高性能

使用
    
    1 下载源码
    2 编译
     make install
    3 分别启动服务
    ./server 12001 2 12000 12002
    ./server 12002 3 12001 12000
    ./server 12000 1 12001 12002
    4等待同步完成
    5客户端链接
     ./client
    6执行操作
     set 【key】 【value】
     
支持类型
    字符串
